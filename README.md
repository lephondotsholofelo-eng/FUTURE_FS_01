# FUTURE_FS_01

## Overview

FUTURE_FS_01 is a personal portfolio website built to showcase my projects, technical skills, and contact information. The application focuses on clean UI design, responsive layout, and professional presentation for potential employers and collaborators.

## Tech Stack

* Next.js
* React
* Tailwind CSS
* Vercel (Deployment)

## Features

* Fully responsive portfolio layout
* Projects section with structured presentation
* Contact section with email interaction
* Navigation between sections using smooth scrolling
* Clean and minimal UI design

## Project Structure

```
/app
/components
/public
/styles
```

## Installation and Setup

1. Clone the repository:

```
git clone https://github.com/lephondotsholofelo-eng/FUTURE_FS_01.git
```

2. Navigate into the project:

```
cd FUTURE_FS_01
```

3. Install dependencies:

```
npm install
```

4. Run the development server:

```
npm run dev
```
